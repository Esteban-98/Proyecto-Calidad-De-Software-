

//Con base en el Script utizando workbench

2.Ir a la pestaña File 
3.opcion Open SQl script 
4.abrir el archivo "ScriptBaseDeDatos_inventario"

//Con base en el Script utizando Cliente MySQL (Command Line Client )
1.Abrir la linea de comando del cliente
2.iniciar sesion local con el usuario y la contraseña
3.abrir el archivo "ScriptBaseDeDatos_inventario" con Bloc de notas 
4.Copiar el contenido del bloc de notas en la linea de comando 
5.base de datos creada
